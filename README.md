# Foodingo

Foodingo is a React + Node food-ordering demo app (local workspace).

Repository: https://github.com/harsh-85293/foodingo

Quick start

1. Install dependencies (root + server):

```bash
npm install
cd server
npm install
```

2. Run development server:

```bash
# in repo root
npm start
# or run server separately
cd server
node server.js
```

Notes

- This repo includes a `server/` folder with Express server, and `src/` frontend React app.
- Add collaborators and enable Codespaces from the repository settings on GitHub.
